# Aka Portfolio Website

Static website structure for GitHub Pages / normal hosting.

## Structure

```text
root/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── img/
│   └── programmer-scene.svg
└── music/
    └── suno_prompt.txt
```

Copy these files/folders into the old repository root. If the old repo already has `css`, `js`, `img`, or `music`, merge the contents carefully.
